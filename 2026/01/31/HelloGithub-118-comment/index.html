clifm

Pure Command-Line File Manager.This is a command-line file management tool written in C. It is more efficient than Shell and lighter than TUI tools. With no complex menus or graphical interfaces, users can quickly perform file operations by entering simple commands, and it supports features like numerical indexing of items, auto-completion, syntax highlighting, and file preview.

  1. Compile from source

    You will get a ELF format file in current working directory named with clifm which has only 1.3M size for now.

  2. Usage

    You can just start it by typing ./clifm command.

  3. Features

    • ELN: You will see the number ahead of directory, the official called it “ELN, Entry List Number”, it’s understandable beacause it is more like display the inode expclitly and you could do almost all of operations with it and i think ELN is the CORE machanism of this file manager project or the attractive parts(the design was applied earlier on nffm project).

    • Trashing: The trash system are supported in this file manager, you will never worry about deleting some files wrongly.

    • Workspace: The workspace machanism was introduced to give you a experience as tmux workspace.

    • Bookmark: Help you with jumping quickly into the directory or file you visited frequently.

    • Tag: Tag files in order to do batch file operations.

  4. Summary

    This is a file manager project, not a shell, but a file manager in shell, focused on file/directory operations.

  5. Rating

    8.0/10.0
